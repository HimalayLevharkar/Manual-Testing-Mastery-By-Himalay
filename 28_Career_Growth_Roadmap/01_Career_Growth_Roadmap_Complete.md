## Career Levels

### Level 1: Junior QA Engineer (0-2 Years)

**Responsibilities:**
```
Testing Activities:
✓ Execute test cases
✓ Report bugs
✓ Basic regression testing
✓ Manual testing
✓ Follow test plans
✓ Document findings
✓ Update test status

Support Activities:
✓ Attend team meetings
✓ Ask for guidance
✓ Learn from seniors
✓ Build domain knowledge
✓ Understand product
```

**Required Skills:**
```
Technical:
- Manual testing expertise
- Testing fundamentals
- Basic SQL
- Basic bug tracking (JIRA)
- Windows/Linux basics
- Browser tools

Soft Skills:
- Communication
- Attention to detail
- Time management
- Teamwork
- Documentation
- Curiosity
```

**Expected Salary (India):**
```
₹2,00,000 - ₹3,50,000 per annum
(Varies by location & company)
```

**Performance Metrics:**
```
✓ Test case execution: 80%+
✓ Bug report quality: Good
✓ Regression coverage: 85%+
✓ Adherence to schedule: 90%+
✓ Defect escape rate: <5%
```

## Certifications

### Recommended Certifications

#### **Level 1: Foundation**

**ISTQB Foundation**
```
Exam: 40 questions in 60 minutes
Pass Percentage: 65%
Duration: 3-6 weeks preparation
Cost: ₹3,000-5,000
Value: High (Industry recognized)
For: Entry-level to mid-level

Topics:
- Testing fundamentals
- Test lifecycle
- Test design techniques
- Test management
- Automation basics
```

#### **Level 2: Intermediate**

**ISTQB Advanced - Test Analyst**
```
Requirement: ISTQB Foundation first
Exam: 50 questions in 90 minutes
Pass Percentage: 60%
Duration: 6-8 weeks preparation
Cost: ₹6,000-8,000
Value: Very High

For: Mid-level testers
Topics:
- Advanced test design
- Testing best practices
- Defect management
- Test organization
```

**ISTQB Advanced - Test Manager**
```
For: QA managers, leads
Topics:
- Test management
- Team leadership
- Project management
- Metrics & KPIs
- Risk management
```

#### **Level 3: Specialist Certifications**

**Selenium WebDriver Certification**
```
For: Automation testers
Topics: Selenium, Page Object Model
Duration: 4-6 weeks
Cost: ₹4,000-6,000
```

**API Testing Certification**
```
For: API testing specialists
Topics: Postman, REST, SOAP
Duration: 3-4 weeks
Cost: ₹3,000-5,000
```

**Performance Testing (LoadRunner/JMeter)**
```
For: Performance engineers
Topics: Load testing, performance analysis
Duration: 6-8 weeks
Cost: ₹6,000-10,000
```

### Certification Roadmap

```
Year 1:
├─ ISTQB Foundation (Months 6-9)
└─ Basic domain certification (Months 10-12)

Year 2-3:
├─ ISTQB Advanced - Test Analyst (Months 12-18)
└─ Specialist certification (Automation/Performance) (Months 18-24)

Year 4-5:
├─ ISTQB Advanced - Test Manager (Months 36-42)
└─ Advanced specialist cert (Months 42-48)

Year 6+:
├─ Domain-specific certifications
├─ Leadership certifications
└─ Advanced technical certifications
```

## Interview Preparation

### Interview Questions by Level

#### **Junior Level (0-2 years)**

```
Q1: What is software testing?
Q2: Explain test case?
Q3: Difference between severity and priority?
Q4: How do you write a bug report?
Q5: SDLC vs STLC?
Q6: Types of testing?
Q7: Regression testing?
Q8: Smoke testing?
Q9: What is test coverage?
Q10: JIRA workflow?

Preparation Time: 2-3 weeks
Focus: Fundamentals, clarity
```

#### **Senior Level (4-7 years)**

```
Q1: Explain risk-based testing?
Q2: How do you optimize test cases?
Q3: Test metrics and KPIs?
Q4: Defect leakage analysis?
Q5: How do you mentor junior testers?
Q6: Process improvement examples?
Q7: Handling tight deadlines?
Q8: Test strategy creation?
Q9: Automation framework design?
Q10: Leadership challenges?

Preparation Time: 3-4 weeks
Focus: Experience, examples, leadership
```

#### **Manager Level (7+ years)**

```
Q1: Your testing vision?
Q2: Team building approach?
Q3: Process transformation?
Q4: Budget management?
Q5: Handling quality vs schedule conflicts?
Q6: Vendor management?
Q7: Organizational restructuring?
Q8: Career development strategy?
Q9: Industry trends knowledge?
Q10: Business impact of testing?

Preparation Time: 4-5 weeks
Focus: Strategy, business, leadership
```

### Interview Preparation Steps

```
Week 1:

- Review job description
- Practice common questions

Week 2:
- Mock interviews
- Technical preparation
- Domain knowledge review
- Weakness addressing

Week 3:
- Final preparation
```
